# News-Press 


- React Typescript
- MUI
- Vite
- React Router DOM
- Font Awesome
- ESLint



### Getting Started.


Make sure you have Node.js and npm

    git clone https://github.com/ddimkou/supercar-rent.git

    cd news-press

    npm install

    Get an API key from https://newsapi.org/

    create a file .env.local

    inside type: VITE_NEWS_API_KEY= [your_api_key]

    npm run dev
